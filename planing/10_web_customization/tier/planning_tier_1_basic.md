# Planning: Tier 1 — Basic Customization

## Tujuan
Membuat 3 customization Tier 1 yang simpel. Perubahan hanya di warna, cahaya, dan elemen visual ringan.

---

## Tier 1 Characteristics
- Perubahan warna tema (primary, secondary, accent)
- Efek cahaya background (glow, gradient)
- Perubahan font atau font weight
- Tidak ada 3D atau animasi berat
- Library: CSS only atau Anime.js ringan

---

## Customization 1: "Golden Empire"
- Warna dominan: emas (#FFD700) + hitam (#1a1a1a)
- Background: gradient emas subtle
- Efek: glow emas pada card hover
- Font: Cinzel (lebih berat)

## Customization 2: "Frozen Tundra"
- Warna dominan: biru es (#A8D8EA) + putih (#F5F5F5)
- Background: gradient biru dingin
- Efek: shimmer putih pada hover
- Font: Outfit (modern clean)

## Customization 3: "Blood Moon"
- Warna dominan: merah gelap (#8B0000) + hitam (#0a0a0a)
- Background: gradient merah gelap
- Efek: glow merah pada card hover
- Font: Cinzel (gothic feel)
